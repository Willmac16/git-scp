# git-scp
Secure copy your changes

## Installation

```
git clone git@github.com:carsonreinke/git-scp.git
cp git-scp /usr/local/bin
```

## Usage

```
git scp <username@hostname> <target-dir> head
git scp <username@hostname> <target-dir> status
```
